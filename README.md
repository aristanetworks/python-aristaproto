# Arista Protobuf / Python gRPC bindings generator & library

> :warning: `aristaproto` 0.x was forked from the original [`betterproto`](https://github.com/danielgtaylor/python-betterproto) repository. This fork is kept in the `v0.x` branch.

> :warning: `aristaproto` 2.x is re-forked from [`betterproto2`](https://github.com/betterproto/python-betterproto2) which is a later fork of the same original.

Changes in this project compared with the base project:

- Renamed to `aristaproto`.

Planned changes:

- Support for precision timestamps (nanoseconds)
- Support for grpcio async bindings (both client and server)
